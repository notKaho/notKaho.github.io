Water Management


A flood control system, a 24-hour weather monitoring system, and a pedestrian lamp post were offered as solutions. The flood control system will utilize an infrared sensor to assess whether the water level is dangerously high, and if it is, it will activate the Relay, which will work as a drainage pump, and send an email to the proper authorities. We'll use the DHT11 to track temperature and humidity for a 24-hour weather monitoring system, then upload the data to Thingspeak Dashboard and do the necessary analysis to forecast and investigate the weather.

How does it work?

Requirements

Requirement 1: When LDR is NOT Triggered (Day), RGB LED will turn OFF.
Requirement 2: When LDR is Triggered (Night), RGB LED will turn ON.
Requirement 3: Relay & Buzzer will turn ON when Avoid Sensor Module is Triggered.
Requirement 4: An email notification will be sent from Raspberry Pi’s code when Requirement 3 is TRUE.
